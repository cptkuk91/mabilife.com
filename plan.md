# Mabi Life Project Plan

## 1. 현재 구현 현황 (Current Status)

### 1.1. 프로젝트 설정
- [x] **Framework**: Next.js (App Router)
- [x] **Styling**: Vanilla CSS (Apple Style Aesthetic)
- [x] **Icons**: Font Awesome

### 1.2. 주요 페이지 및 기능
- [x] **메인 페이지 (`/`)**
  - [x] Apple 스타일 히어로 섹션
  - [x] Bento Grid 레이아웃의 대시보드
  - [x] 검색창 (클릭 시 검색 페이지로 이동)
- [x] **공략 라이브러리 (`/guide`)**
  - [x] 카테고리별 아이콘 그리드
  - [x] 가로 스크롤(Horizontal Scroll) 형태의 추천 공략 섹션
  - [x] 실시간 유저 팁 리스트
- [x] **커뮤니티 (`/community`)**
  - [x] Q&A와 광장(자유게시판) 통합
  - [x] 탭 필터링 (전체, 잡담, Q&A, 정보)
  - [x] 글쓰기 UI 및 게시물 리스트
- [x] **검색 결과 (`/search`)**
  - [x] 통합 검색 UI
  - [x] Top Hit (위키 스타일) 카드 및 검색 결과 리스트
- [x] **공통 컴포넌트**
  - [x] **Navbar**: 페이지별 활성화 상태 표시, 로그인 버튼 포함

## 2. 향후 작업 계획 (Next Steps)

### 2.1. 우선 순위 (High Priority)
- [ ] **로그인 페이지 구현 (`/login`)**
    - [ ] 소셜 로그인 버튼 (구글, 카카오, 네이버 등)
    - [ ] 이메일/비밀번호 입력 폼
    - [ ] 회원가입 링크
    - [ ] 디자인: Glassmorphism을 활용한 깔끔한 모달 또는 전용 페이지

- [ ] **상세 페이지 구현 (Detail Pages)**
    - [ ] 공략 상세 (`/guide/[id]`)
    - [ ] 커뮤니티 게시글 상세 (`/community/[id]`)
    - [ ] 기능: 본문 읽기, 댓글 작성/보기, 좋아요 기능

### 2.2. 중기 계획 (Medium Priority)
- [ ] **글쓰기 기능 고도화 (`/write`)**
    - [ ] 긴 글(공략 등)을 작성할 수 있는 전용 에디터 페이지
    - [ ] 이미지 업로드, 텍스트 서식 적용 등

- [ ] **프로필 페이지 (`/profile`)**
    - [ ] 유저 정보, 작성한 글 목록, 스크랩한 공략 확인
    - [ ] 내 캐릭터 정보 연동 (마비노기 API 활용 가능성 검토)

### 2.3. 기능 및 백엔드 (Backend & Logic)
- [ ] **데이터 연동**
    - [ ] 더미 데이터 제거 및 실제 DB/API 연동
    - [ ] Supabase 또는 Firebase 도입 검토

- [ ] **상호작용 기능**
    - [ ] 좋아요, 댓글, 스크랩 등 실제 인터랙션 구현

---

## 3. 제안 사항 (Suggestion)
- **로그인 페이지**를 먼저 작업하여 사용자 흐름(Flow)을 완성하는 것을 추천합니다.
- 이후 **게시글 상세 페이지**를 만들어 리스트에서 클릭했을 때의 경험을 연결하는 것이 좋습니다.
